# Power On / Off

This document explains how to start up or shut down the **Indevolt SolidFlex / PowerFlex 2000** energy storage device using the physical button.

## Power On

You may need to power on the device in the following situations:

- After installation or rewiring is completed  
- When the system has been manually shut down and needs to be restarted  
- After a long power outage has been restored  

**Steps:**

1. Confirm that the device is currently powered off.  
2. Press and hold the power button for about 2 seconds.  
3. Release the button, and the device will start up with the green indicator light on.  
4. The blue LED strip will display an animation spreading outward from the center.  
5. After startup is complete, the LED strip will display the corresponding system status indicator. See [LED Indicators](../overview/appearance-interface.md#led-indicators).

## Power Off

It is recommended to shut down the device first in the following situations:

- Device maintenance, cleaning, or wiring operations are required  
- The system will not be used for an extended period  
- External power sources need to be safely disconnected  

**Steps:**

1. Confirm that the device is currently in powered-on state.  
2. **Disconnect the PV input connection and wait for more than 20 seconds** to ensure the system stops PV power input.  
3. Press and hold the power button for about 2 seconds.  
4. The blue LED strip will display an animation converging from both sides toward the center.  
5. Release the button and the system will begin the shutdown process.  
6. When the indicator light is completely off, the device has successfully shut down.

:::info

- Do not press the power button repeatedly during startup or shutdown.  
- If maintenance or wiring work is required after shutdown, wait until the device has completely stopped operating.  
- If the device will not be used for a long time, it is recommended to shut it down and disconnect external power sources.
:::
