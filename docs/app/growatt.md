# How to get Growatt API token

In the ShinePhone app，go to Me (bottom right) > account name > API Token. Create and copy the token by pressing the copy icon on the right.

<img src={require("./img/growatt_token.png").default} />

In the INDEVOLT App, paste the token and choose the right data center.

After your growatt device is online, it might take 5-10 minutes before you can see the devicesn under the growatt device list.
