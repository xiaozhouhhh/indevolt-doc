# Solax

You need tokenID, API Address and Registration number from SolaxCloud to add Solax solar inverter to INDEVOLT app.

## How to get tokenID and API Address?

**Using the Solax app**:
1. In the app, tap **Service** at the bottom of the main screen. On this page, select **Third-party ecosystem**;
2. Choose **SolaxCloud API**;
3. Copy the **token ID** and **API Address** from the app.

   <img src={require("./img/third_party.png").default}  width="240"/>
   <img src={require("./img/solaxcloud_api.png").default}  width="240"/>
   <img src={require("./img/tokenid.png").default}  width="240"/>



**Using Solax web**:
1. On the main page, go to **Service** in the left sidebar;
2. Click **Third-party ecosystem**;
3. Here you can copy your **token ID** and **API Address**. 

   <img src={require("./img/solax_web_third_party.png").default}/>
   <img src={require("./img/solax_web_tokenid.png").default}/>


:::note
If no Token ID is displayed, click **Get** to request one. Then return to step 3 to copy it.

<img src={require("./img/get_tokenid.png").default} width="480"/>
:::

## How to find Registration number?

**Using the Solax app**:
1. In the app, tap **Device** at the bottom of the main screen;
2. Your inverter will be listed with the **Registration Number** shown below it.

   <img src={require("./img/app_registration_number.png").default}  width="240"/>


**Using Solax web**:
1. On the main screen, go to **Devices** in the left sidebar;
2. In the **Devices list**, you will find the **Registration number**.

   <img src={require("./img/web_registration_number.png").default}/>
