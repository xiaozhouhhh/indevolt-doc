# How to Get KeyID and KeySecret

To connect SolisCloud with INDEVOLT, you need to provide the KeyID and KeySecret from SolisCloud.

Please refer to the section **"2. API to Obtain Single Account Permission"** in the [official Solis documentation](https://doc.soliscloud.com/en/20.API%20documentation/01.SolisCloud%20Platform%20API%20Document.html) for instructions.
