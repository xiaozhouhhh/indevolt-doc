# Modèles de produits

Ce document présente les principaux modèles de produits suivants :

- SolidFlex 2000  
- SolidFlex 2000Eco  
- PowerFlex 2000  
- PowerFlex 2000Eco  
- SFA1800  
- PFA2000  
- SolidFlex 1200  
- SolidFlex 3000 Hybrid  
- SolidFlex 3000 AC  
- PowerFlex 3000 Hybrid  
- PowerFlex 3000 AC  
- SFA3600  
- PFA4000
