# Consultation des états et des données

Vous pouvez à tout moment consulter, dans l’application INDEVOLT, l’état de fonctionnement de l’appareil ainsi que les données de production solaire, de stockage et de consommation du foyer.

- **Consulter l’état de fonctionnement de l’appareil**  
  Après avoir ajouté un appareil dans l’application, appuyez sur la carte correspondante pour accéder à la [page de détails de l’appareil](../../app/device.md#2-détails-de-lappareil). Vous pouvez y consulter l’état de connexion, le mode de fonctionnement et la puissance actuelle.

- **Consulter les données de production et de consommation**  
  Dans la barre de navigation inférieure de l’application, appuyez sur [Rapport](../../app/report.md) pour accéder à la page de synthèse énergétique du foyer. Vous pouvez y consulter les tendances de consommation (jour / semaine / mois / année), la production photovoltaïque ainsi que l’estimation des économies réalisées.
