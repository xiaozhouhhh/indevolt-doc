# FAQ

## « L'utilisateur n'existe pas » ou « Erreur de compte ou de mot de passe » lors de la connexion

1. Assurez-vous que le compte que vous avez saisi (email/numéro de téléphone) est correct.  
2. Vérifiez si votre mot de passe est sensible à la casse. Si vous avez oublié votre mot de passe, appuyez sur **« Mot de passe oublié »** sur la page de connexion pour le réinitialiser.

## Que faire si je ne reçois pas le code de vérification ?

1. Si vous utilisez la vérification par téléphone, vérifiez si le blocage ou le filtrage des SMS est activé sur votre appareil.  
2. Si vous utilisez la vérification par email, vérifiez votre dossier spam/courrier indésirable au cas où le message aurait été filtré incorrectement.

## Comment supprimer mon compte ?

La suppression du compte est irréversible. Veuillez procéder avec prudence. Pour les étapes détaillées, consultez [Suppression du compte](./profile.md#13-supprimer-le-compte).

## Que faire si mon appareil apparaît hors ligne ?

1. Vérifiez que l'appareil est sous tension, que les voyants sont normaux et que le réseau Wi-Fi auquel il est connecté fonctionne correctement.  
2. Sur la page Appareil, appuyez longuement sur la carte de l’appareil et appuyez sur **WLAN** pour reconfigurer le réseau : voir **[Configuration réseau](./add-device.md#étape-3--configurez-le-réseau)**.  
3. Si la configuration échoue, redémarrez l'appareil : éteignez-le complètement, attendez 30 secondes, puis rallumez-le.
