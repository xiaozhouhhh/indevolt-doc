# Comment obtenir la KeyID et le KeySecret

Pour connecter SolisCloud à INDEVOLT, vous devez obtenir la KeyID et le KeySecret depuis SolisCloud.

Veuillez consulter la section **"2. API to Obtain Single Account Permission"** de la [documentation officielle de Solis](https://doc.soliscloud.com/en/20.API%20documentation/01.SolisCloud%20Platform%20API%20Document.html) pour obtenir les instructions.
