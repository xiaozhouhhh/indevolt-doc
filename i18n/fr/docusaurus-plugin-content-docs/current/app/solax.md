# Solax

Pour ajouter votre onduleur solaire Solax à l’application INDEVOLT, vous aurez besoin de **tokenID**, **API Address** et **Registration Number** depuis SolaxCloud.

---

## Comment obtenir tokenID et API Address ?

**Avec l’application Solax** :
1. Dans l’application, appuyez sur **Service** en bas de l’écran principal, puis sélectionnez **Third-party ecosystem** ;
2. Choisissez **SolaxCloud API** ;
3. Copiez le **token ID** et **API Address** affichés dans l’application.

   <img src={require("./img/third_party.png").default}  width="240"/>
   <img src={require("./img/solaxcloud_api.png").default}  width="240"/>
   <img src={require("./img/tokenid.png").default}  width="240"/>

**Avec Solax Web** :
1. Sur la page principale, allez dans **Service** dans la barre latérale gauche ;
2. Cliquez sur **Third-party ecosystem** ;
3. Vous pouvez ici copier votre **token ID** et **API Address**.

   <img src={require("./img/solax_web_third_party.png").default}/>
   <img src={require("./img/solax_web_tokenid.png").default}/>

:::note
Si aucun Token ID n’est affiché, cliquez sur **Get** pour en demander un, puis retournez à l’étape 3 pour le copier.

   <img src={require("./img/get_tokenid.png").default} width="480"/>
:::

---

## Comment trouver le Registration Number ?

**Avec l’application Solax** :
1. Dans l’application, appuyez sur **Device** en bas de l’écran principal ;
2. Votre onduleur sera listé avec le **Registration Number** affiché dessous.

   <img src={require("./img/app_registration_number.png").default}  width="240"/>

**Avec Solax Web** :
1. Sur la page principale, allez dans **Devices** dans la barre latérale gauche ;
2. Dans la **Devices List**, vous trouverez le **Registration Number**.

   <img src={require("./img/web_registration_number.png").default}/>
