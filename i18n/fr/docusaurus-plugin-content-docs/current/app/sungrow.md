# Comment obtenir l’Appkey et la Secret key Sungrow

Suivez les étapes ci-dessous pour créer une application dans le portail développeur Sungrow et obtenir votre **Appkey** et votre **Secret Key**.

1. Accédez à https://web3.isolarcloud.eu/#/login et connectez-vous à votre compte iSolarCloud.

   <img src={require("./img/sungrow1.png").default} />

2. Cliquez sur l’icône <img src={require("./img/sungrow_icon.png").default} width="20" style={{verticalAlign: "middle"}}/>, située dans l’angle supérieur droit, puis sélectionnez **Developer portal**.

   <img src={require("./img/sungrow2.png").default} />

3. Une fois dans le [Developer Portal](https://developer-api.isolarcloud.com/#/home), cliquez sur **Applications** dans l’angle supérieur droit pour ouvrir la page **My applications**.

   <img src={require("./img/sungrow3.png").default} />

4. Cliquez sur **Create application** pour créer une nouvelle application.

   <img src={require("./img/sungrow4.png").default} />

5. Renseignez les informations de l’application :

   - **Application name** : saisissez le nom de l’application (INDEVOLT) ou un autre nom facilement identifiable, par exemple `My Home EMS`.
   - **Company name** : saisissez le nom du domicile créé dans l’application ou un autre identifiant approprié, par exemple `My Home`.
   - **Application description** : saisissez une brève description, par exemple `Data Integration`.
   - **Authorize with OAuth2.0** : sélectionnez **No**.

   Cliquez ensuite sur **Create application**.

   <img src={require("./img/sungrow5.png").default} />

6. Revenez à la page **My applications**, puis cliquez sur **View** pour l’application que vous venez de créer.

   <img src={require("./img/sungrow6.png").default} />

7. Votre **Appkey** et votre **Secret key** s’affichent sur la page des détails de l’application.

   <img src={require("./img/sungrow7.png").default} />
