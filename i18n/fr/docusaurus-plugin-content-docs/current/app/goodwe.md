# Comment se connecter à GoodWe


## Étape 1 : Compte SEMS requis

Pour accéder à l’API SEMS de GoodWe, vous devez disposer d’un compte SEMS (adresse e-mail/nom d’utilisateur et mot de passe).

Si vous n’avez pas encore de compte SEMS, veuillez d’abord en créer un.

---

## Étape 2 : Configurer la source de données

Cliquez sur « Add access » et saisissez les informations suivantes :

- Nom d’utilisateur : Adresse e-mail du compte SEMS
- Mot de passe (non chiffré) : Mot de passe du compte SEMS

Une fois votre appareil en ligne, il peut être nécessaire d’attendre 5 à 10 minutes avant que le numéro de série (SN) apparaisse dans la liste des appareils GoodWe.
