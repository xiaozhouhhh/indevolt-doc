# 家庭

**家庭**页面是您管理当前家庭能源数据的主界面，集中展示家庭的实时发电、用电情况等数据及能量流动全景。

<img src={require("./img/home_intro.png").default} width="240"/>


| 序号 | 功能         | 说明                                                                                         |
| ---- | ------------ | -------------------------------------------------------------------------------------------- |
| 1    | 家庭     | 显示当前家庭名称，点击可查看并切换已添加的所有家庭，以及管理家庭。                           |
| 2    | +            | 添加设备。                                                                                   |
| 3    | 看板数据配置 | 自定义能量流动图的显示主题与数据指标。                                                       |
| 4    | 天气         | 显示当地实时天气状况与温度。                                                                 |
| 5    | 去配置       | 当无电价方案时，显示去配置按钮，点击后可设置购电与售电价格；有方案时，则直接显示当天的电价。 |
| 6    | 家庭流动图   | 以可视化图形动态展示家庭内部的能量来源与去向。                                               |
| 7    | 日数据          | 以日为单位的发电与用电数据视图。                                                     |


## 1. 我的家庭

### 1.1 切换家庭

点击界面左上角的家庭名称，即可展开账号下所有的家庭列表。选择目标家庭，即可快速切换至该家庭的能源数据视图。

<img src={require("./img/tap_myhome.png").default} width="240"/>
<img src={require("./img/select_home.png").default} width="240"/>

### 1.2 家庭管理

如需对家庭进行详细管理，可点击列表下方的**家庭管理**按钮进入管理页面。
<img src={require("./img/select_home.png").default} width="240"/>
<img src={require("./img/manage_home.png").default} width="240"/>

**添加家庭**

1. 在家庭管理页面点击右上角的 **+** 按钮。
2. 根据页面指引，完成新家庭的信息录入。具体步骤请参考[添加家庭](./profile.md#21-添加家庭)章节。


**编辑家庭**

在家庭管理页面选择需要修改的家庭，即可进入其信息页进行编辑：
- [编辑](./profile.md#22-编辑家庭信息)家庭名称、地址及相关配置。
- [删除](./profile.md#23-删除家庭)该家庭。

## 4. 天气

此组件用于展示您家庭所在地的实时天气信息与温度。

## 5. 去配置

此区域动态显示内容：
- 当当天无电价方案时，显示**去配置**按钮，点击后可[设置购电与售电价格](./profile.md#3-电价设置)；
- 当当天有电价方案时，则显示当天的购电与售电电价，点击可查看历史电价。

## 6. 家庭流动图

家庭流动图以可视化方式，动态呈现您家庭内部的能量来源与实时去向。图中能源移动方向即代表能量的实际流动方向。
- 刷新视图：在流动图页面下拉，即可刷新数据。
- 主题设置：如需更改显示主题，可点击[看板数据配置](#3-看板数据配置)进行调整。
- 设置数据源：流动图所展示的数据来源于[数据源设置](./profile.md#5-数据源设置)，您可根据需求进行调整。

<img src={require("./img/power_flow.png").default} width="240"/>


## 7. 日数据

日数据视图集中展示当日各项发电与用电的累计电量数据：

| 参数     | 说明                                                                                 |
| -------- | ------------------------------------------------------------------------------------ |
| 自发自用 | (光伏供电的用电量 + 光伏对电池的充电量) / 发电量，衡量光伏发电被家庭自身利用的比例。 |
| 自给自足 | (来自发电的用电量 + 来自电池放电的用电量) / 用电量，衡量家庭用电中自产能源的占比。   |
| 发电量   | 光伏系统当日的总发电量。                                                             |
| 用电量   | 家庭当日消耗的总电量。                                                               |
| 购电量   | 当日从电网购买的总电量。                                                             |
| 售电量   | 当日向电网输送的总电量。                                                             |
| 充电量   | 储能设备当日充电总量。                                                               |
| 放电量   | 储能设备当日放电总量。                                                               |


<img src={require("./img/daily.png").default} width="240"/>

您可通过[看板数据配置](#3-看板数据配置)，自定义此处显示的数据指标。
