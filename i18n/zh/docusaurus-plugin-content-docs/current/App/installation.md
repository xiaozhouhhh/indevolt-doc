# App安装


:::info 安装准备

在开始安装前，请确保您的设备满足以下条件:
- 操作系统：iOS 13.0及以上；Android 5.0及以上。
- 已连接可用的 WLAN 或蜂窝网络；
- 有足够的剩余存储空间；
- 设备电量充足，或已连接至电源。

:::
