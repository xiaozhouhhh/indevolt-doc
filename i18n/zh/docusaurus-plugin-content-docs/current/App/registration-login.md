# 账户注册与登录

INDEVOLT App安装完成后，打开App进入登录页。

<img src={require("./img/login_page.png").default} alt="Login page" width="240"/>


## 新用户注册

请按照以下步骤创建您的新账户。

### 步骤 1、进入注册页面

在应用登录页，点击**快速注册** 按钮。
<img src={require("./img/register.png").default} width="240"/>

### 步骤 2、填写注册信息

根据页面提示，填写必要的注册信息。


1. 输入您常用的电子邮箱地址。
2. 点击**获取验证码**。
3. 拖动滑块验证邮箱。
4. 输入您收到的邮箱验证码。
5. 设置一个安全的密码（至少8位，且至少包含以下3种字符：数字、大小写字母、符号）。

   <img src={require("./img/email_register1.png").default} width="240"/>
   <img src={require("./img/email_register2.png").default} width="240"/>

:::info 该账户已注册

若提示该账户已注册，则说明该账号已存在，需按照[账户登录](#账户登录)部分使用密码登录或通过[忘记密码](#忘记密码)找回密码。

:::


:::warning

出于账号安全考虑，密码不能过于简单，如连续的阿拉伯数字“123456”或同一字符多次重复的“AAAAA123”都将被禁止使用。

:::

### 步骤 3、完成注册

1. 阅读并勾选同意《服务协议》与《隐私政策》。
2. 确认信息无误后，点击**注册**按钮。

系统验证成功后，App自动跳转至应用主界面。

---

## 账户登录

当您已拥有账户后，请通过以下方式登录。

1. 通过点击**更多登录方式**按钮切换登录方式，支持手机号、邮箱、用户名登录。
2. 输入您注册时使用的手机号/邮箱/用户名。
3. 输入您设置的密码。
4. 阅读并勾选同意《服务协议》与《隐私政策》。
5. 点击**登录**按钮。

   <img src={require("./img/login_methods.png").default} width="240"/>
   <img src={require("./img/login_page.png").default} alt="Login page" width="240"/>


登录成功后，App自动跳转至应用主界面。

---

## 忘记密码

1. 在登录页面点击**忘记密码**按钮；
2. 选择通过邮箱或手机找回；
3. 填写注册时绑定的邮箱或手机号接收验证码；
4. 点击**获取验证码**；
5. 拖动滑块验证邮箱/手机号；
6. 输入收到的邮箱或短信验证码；
7. 点击**下一步**；
8. 在重置密码页输入新的密码和确认密码，点击**确认**按钮；
   
   <img src={require("./img/forget_password.png").default} alt="Login page" width="240"/>
   <img src={require("./img/confirm_account1.png").default} alt="Confirm account page" width="240" />
   <img src={require("./img/confirm_account2.png").default} alt="Confirm account page" width="240"/>
   <img src={require("./img/reset_password.png").default} alt="Reset password page" width="240"/>

密码重置成功，单击**登录**后回到登录页面重新登录即可。
