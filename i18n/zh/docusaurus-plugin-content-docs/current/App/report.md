# 统计

统计页面为您提供全面的家庭能源数据分析，支持从**日、周、月、年**四个时间维度查看能源数据。您可通过顶部选项卡切换维度，并调整下方的时间段，页面将动态展示对应周期内的聚合数据。

<img src={require("./img/report.png").default} width="240"/>

## 1. 数据指标说明

| 参数         | 说明                                                        |
| ------------ | ----------------------------------------------------------- |
| **发电量**   | 光伏系统在选定周期内产生的总电量。                          |
| **用电量**   | 家庭在选定周期内消耗的总电量，包含光伏、电池与电网供电。    |
| **电网购电** | 从公共电网获取的总电量。                                    |
| **并网售电** | 向公共电网输送的总电量。                                    |
| **放电量**   | 储能系统在周期内释放的总电量。                              |
| **充电量**   | 储能系统在周期内吸收并存储的总电量。                        |
| **发电功率** | 图表选中时间点光伏系统的实时发电功率。                      |
| **充电功率** | 图表选中时间点储能电池的实时充电功率。                      |
| **放电功率** | 图表选中时间点储能电池的实时放电功率。                      |
| **SOC**      | 图表选中时间点储能电池的剩余电量百分比（State of Charge）。 |
| **用电功率** | 图表选中时间点家庭所有电器的实时总耗电功率。                |
| **购电功率** | 图表选中时间点从公共电网获取的实时功率。                    |
| **售电功率** | 图表选中时间点向公共电网输送的实时功率。                    |
| **收益**     | 根据电价方案计算的预估收益，具体公式见下文。                |
| **环境贡献** | 展示累计减碳量、等效植树量及节约煤炭量等环保指标。          |

### 1.3 收益

收益由自用节省与售电收益两部分组成，基于您设置的家庭[电价](./profile.md#3-电价设置)方案计算得出。


1. <u>**自用节省** = 节约电量 × 购电电价</u>  
   
    节约电量根据数据可用性采用以下方式计算：
      - 当用电量和购电量可统计时：节约电量 = 总用电量 - 电网购电量
      - 当相关数据不可用时：节约电量 = 发电量 + 电池放电量 - 电池充电量

2. <u>**售电收益** = 售电量 × 售电电价</u>
    
    若无售电行为，则售电收益为0。

3. <u>**总收益** = 自用节省 + 售电收益</u>

您可点击收益数据区域进入收益统计页，支持：
- 查看**日/周/月/年**维度的收益数据。
- 切换**总收益**与**售电收益**视图。
- 查看计算依据的电价方案，点击可跳转至电价配置页面。

## 2. 功能操作

### 2.1 高级模式

点击右上角<img src={require("./img/config_icon.png").default} width="25" style={{verticalAlign: "middle"}}/>按钮，可切换至**高级模式**，查看更直观的能源流向图：

<img src={require("./img/mode_selection.png").default} width="240"/>
<img src={require("./img/advanced_mode.png").default} width="240"/>

### 2.2 数据视图

每个时间维度均提供**概览**数据，以及**用电**、**光伏**、**电池**、**电网**的详细分类数据。通过点击对应图标，即可在不同数据视图间切换。

<img src={require("./img/report_overview.png").default} width="240"/>

### 2.3 图表显示控制

在曲线图区域上方，点击各数据指标对应的按钮，可**显示或隐藏**该数据线在图表中的显示，便于您进行针对性分析。
