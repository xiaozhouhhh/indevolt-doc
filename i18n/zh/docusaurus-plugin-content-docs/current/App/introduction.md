# App简介

INDEVOLT 是一款专为家庭用户设计的智能能源管理应用，致力于帮助您更高效、更经济地管理家庭能源。通过连接光伏、储能、充电桩等各类能源设备，我们为您提供一站式的能源监控、优化与管理体验。

<img src={require("./img/app_overview.png").default} alt="App overview" width="240"/>

**核心功能**

🏠 家庭能源全景视图

- 实时掌握发电、用电、储能动态
- 直观展示能源流动与数据统计
- 支持多家庭管理，随时切换查看

🔌 智能设备连接

- 兼容主流品牌能源设备
- 支持设备直连与远程控制
- 实时监控设备状态与运行数据

📊 多维数据洞察

- 提供日、周、月、年多维度数据分析
- 清晰展示用电趋势与节能空间
- 生成个性化能源报告

💡 智能优化管理

- 基于分时电价自动优化用能策略
- 支持自定义充放电计划
- 智能场景模式，最大化能源效益

🌱 环保贡献可视化

- 实时计算减碳量与环保贡献
- 展示节能收益与环境效益

🔧 专业运维支持

- 远程调试功能，快速解决设备问题
- 完善的设备管理与故障排查
- 专业的技术支持服务

只需几步简单操作，即可[开启智能能源管理之旅](./installation.md)。
