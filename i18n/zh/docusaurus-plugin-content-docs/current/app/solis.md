# 如何获取 KeyID 和 KeySecret

要将 SolisCloud 连接至 INDEVOLT，您需要提供从 SolisCloud 获取的 **KeyID** 和 **KeySecret**。

请参考 [Solis 官方文档](https://doc.soliscloud.com/en/20.API%20documentation/01.SolisCloud%20Platform%20API%20Document.html) 中的 **"2. API to Obtain Single Account Permission"** 章节获取操作说明。
