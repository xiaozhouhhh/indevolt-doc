# 家庭

**家庭**页面是您查看和管理家庭能源数据的首页。在这里，您可以实时了解家庭的发电、用电、电池充放电以及电网购售电情况，并通过能量流动图直观查看能源的流向。

<img src={require("./img/home_intro.png").default} width="240"/>


| 序号 | 功能         | 说明                                                             |
| ---- | ------------ | ---------------------------------------------------------------- |
| 1    | 家庭     | 显示当前家庭名称，点击可查看并切换已添加的所有家庭，以及管理家庭 |
| 2    | +            | 添加设备                                                         |
| 3    | 天气         | 查看家庭所在地的实时天气和温度                                   |
| 4    | 家庭流动图   | 以可视化图形动态展示家庭内部的能量来源与去向                     |
| 5    | 看板数据配置 | 自定义能量流动图的显示主题与数据指标                             |
| 6    | 日数据       | 查看当天的发电、用电及收益统计                                   |


## 1. 我的家庭

如果您的账号管理多个家庭，可以随时在这里切换查看不同家庭的数据。

### 1.1 切换家庭

点击左上角的家庭名称，即可查看账号下已创建的所有家庭。选择目标家庭后，页面将切换至对应家庭的能源数据视图。

<img src={require("./img/tap_myhome.png").default} width="240"/>
<img src={require("./img/select_home.png").default} width="240"/>

### 1.2 家庭管理

点击家庭列表底部的 **家庭管理**，可进入家庭管理页面。

#### 添加家庭

1. 在家庭管理页面点击右上角的 **+** 按钮。
2. 按照页面提示填写家庭信息。

详细操作请参考：[添加家庭](./profile.md#21-添加家庭)

#### 编辑或删除家庭

在家庭管理页面选择需要修改的家庭，即可进入其信息页进行以下操作：
- [编辑](./profile.md#22-编辑家庭信息)家庭名称、地址及相关配置。
- [删除](./profile.md#23-删除家庭)该家庭。

<img src={require("./img/select_home.png").default} width="240"/>
<img src={require("./img/manage_home.png").default} width="240"/>


## 2. 添加设备

点击首页右上角 **+** 按钮即可开始添加设备。

如需详细操作说明，请参考：[添加设备](./add-device.md)。

## 3. 天气

天气卡片会显示家庭所在地的实时天气和当前温度，帮助您结合天气情况了解能源生产和消耗状况。

## 4. 家庭流动图

家庭流动图以可视化方式，动态呈现您家庭内部的能量来源与实时去向。图中能源移动方向即代表能量的实际流动方向。
- 刷新视图：在家庭页面下拉即可刷新数据。
- 设置主题：如需切换流动图样式，可通过[看板数据配置](#5-看板数据配置)进行设置。
- 设置数据源：流动图所展示的数据来源于[数据源设置](./profile.md#5-数据源设置)，您可根据需求点击对应字段进入设置。

<img src={require("./img/power_flow.png").default} width="240"/>

## 5. 看板数据配置

点击<img src={require("./img/config_icon.png").default} width="25" style={{verticalAlign: "middle"}}/>按钮进入配置页面，自定义首页展示内容。

您可以：

- 选择流动图主题
- 控制首页显示哪些日数据卡片
- 调整数据卡片的显示顺序

<img src={require("./img/select_dashboard.png").default} width="240"/>
<img src={require("./img/dashboard.png").default} width="240"/>


### 5.1 流动图主题

点击 **流动图** 可选择不同的流动图主题样式。

<img src={require("./img/configure_power_flow.png").default} width="240"/>

### 5.2 日数据显示

您可以选择首页显示哪些[日数据](#6-日数据)卡片。

#### 自动

系统会根据当前已配置的数据源自动显示相关数据卡片。

如果某项数据没有对应数据源，则不会显示该卡片。

<img src={require("./img/configure_daily1.png").default} width="240"/>

#### 手动

手动勾选需要显示的数据项目。

<img src={require("./img/configure_daily2.png").default} width="240"/>

### 5.3 调整数据顺序

长按并拖动数据卡片，可自由调整各项数据的显示顺序。调整后，家庭首页的日数据区域将按照新的顺序进行展示，方便将常用数据放在更靠前的位置。

<img src={require("./img/data_order.png").default} width="240"/>



## 6. 日数据

日数据区域用于统计当天截至当前时刻的能源数据，帮助您了解家庭的能源使用情况。

<img src={require("./img/daily.png").default} width="240"/>

| 数据项   | 说明                                                                                 |
| -------- | ------------------------------------------------------------------------------------ |
| 收益     | 根据当前电价计算的当日[收益](./report.md#13-收益)，点击电价卡片可查看详细收益统计    |
| 发电量   | 光伏系统当天累计发电量                                                               |
| 电网     | 当天累计购电量和售电量                                                               |
| 用电量   | 家庭当天累计用电量                                                                   |
| 电价     | 当前设置的购电电价和售电电价                                                         |
| 电池     | 电池当天累计充电量和放电量                                                           |
| 自发自用 | (光伏供电的用电量 + 光伏对电池的充电量) / 发电量，衡量光伏发电被家庭自身利用的比例。 |
| 自给自足 | (来自发电的用电量 + 来自电池放电的用电量) / 用电量，衡量家庭用电中自产能源的占比。   |

您可通过[看板数据配置](#5-看板数据配置)自定义此处显示的数据指标。
