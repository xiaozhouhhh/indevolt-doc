# Solax

要将 Solax 光伏逆变器添加到 INDEVOLT App，您需要从 SolaxCloud 获取 **tokenID**、**API Address** 和 **Registration number**。

---

## 如何获取 tokenID 和 API Address？

**使用 Solax App**：
1. 打开 App，在主屏幕底部点击 **Service**，在此页面选择 **Third-party ecosystem**；
2. 选择 **SolaxCloud API**；
3. 复制 App 中显示的 **token ID** 和 **API Address**。

   <img src={require("./img/third_party.png").default}  width="240"/>
   <img src={require("./img/solaxcloud_api.png").default}  width="240"/>
   <img src={require("./img/tokenid.png").default}  width="240"/>

**使用 Solax Web**：
1. 在主页左侧菜单点击 **Service**；
2. 点击 **Third-party ecosystem**；
3. 在此处复制 **token ID** 和 **API Address**。

   <img src={require("./img/solax_web_third_party.png").default}/>
   <img src={require("./img/solax_web_tokenid.png").default}/>

:::note
如果未显示 Token ID，请点击 **Get** 申请，然后返回步骤 3 复制。

   <img src={require("./img/get_tokenid.png").default} width="480"/>
:::

---

## 如何查看注册编号？

**使用 Solax App**：
1. 打开 App，在主屏幕底部点击 **Device**；
2. 您的逆变器列表中会显示 **Registration Number**。

   <img src={require("./img/app_registration_number.png").default}  width="240"/>

**使用 Solax Web**：
1. 在主页左侧菜单点击 **Devices**；
2. 在 **Devices list** 中找到 **Registration Number**。

   <img src={require("./img/web_registration_number.png").default}/>
