# 寄存器说明

无论使用 Modbus TCP 还是 Modbus RTU，寄存器地址、功能码及数据格式完全一致，仅通信方式不同。

储能系统内部模块（如 EMS、BMS 等）的数据会被映射成 Modbus 寄存器：

:::tip 寄存器地址说明

不同 Modbus 工具对寄存器地址的显示方式可能不同。有些工具使用：
- 从 `0` 开始的实际地址
- 从 `1` 开始的地址
- 带 `40001` 前缀的保持寄存器地址

因此，同一个寄存器在不同软件中，显示方式可能不同。

:::
