# 术语表

## 储能主机

系统的核心单元，内部集成逆变器与控制模块，负责电能转换、能量调度和安全保护。

## 电池模块

用于存储电能的模块化电池单元，可根据需求进行堆叠扩展容量。

## 防逆流

通过限制馈网功率，避免电能反向送入公共电网的功能。

## 负载

家庭中正在用电的设备或用电总需求，如电器、照明等。

## 负载曲线

用户为一天中不同时间段设定的目标功率曲线。

## 关键负载

通过旁路接口连接的重要用电设备，系统仅保障该部分负载供电，不向电网放电。


## 光伏组件

将太阳能转换为电能的发电组件，通常安装在阳台、屋顶或露台等有阳光的位置。

## 价格策略

系统根据用户设置的电价阈值，自动判断何时充电、放电或保持静止，以降低整体用电成本。


## 馈网

设备向公共电网输出电能的行为。


## 逆变器

将电池或光伏产生的直流电（DC）转换为家庭可使用的交流电（AC）的核心部件。


## 逆变器最大输入功率

设备允许的最大充电功率，影响电池从光伏或电网充电的速度。

## 实时控制

允许用户手动、即时控制设备的充电、放电或待机状态。

## 应急电量
 
在停电时为关键设备预留的电量。

## 最大交流输出功率

设备允许的最大放电功率。

## 自定义充放计划

用户可按照设定的时间段，明确指定系统在特定时间充电、放电或保持静止。

## 自用优先

系统优先使用光伏发电为家庭负载供电，多余电量用于给电池充电，尽量减少从电网购电或向电网馈电。

## Digest 认证

一种网络身份验证方式，通过加密机制避免明文传输密码，提升本地 API 调用的安全性。

## OpenData

设备提供的本地 HTTP 接口，允许用户或第三方系统获取设备数据并发送控制指令。

## SoC

State of Charge，电池电量，表示电池当前剩余电量的百分比，范围为 0%–100%。
