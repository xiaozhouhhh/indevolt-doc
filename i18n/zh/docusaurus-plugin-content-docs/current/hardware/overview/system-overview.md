# 系统组成及原理说明

Indevolt 储能是一套面向家庭和小型用能场景的智能储能解决方案，适合以下用户：

- 家中空间有限，但希望使用储能设备  
- 已安装或计划安装光伏，想把白天的电留到晚上用  
- 尚未安装光伏，但希望通过分时电价降低电费  
- 希望更清楚地掌控家里的用电情况  

简单来说，它的作用就是：把电“存起来”，在更合适的时间帮你用掉。

## 系统是如何工作的？

Indevolt 储能系统目前包含多款型号，不同型号在接口配置上有所差异，适合的使用场景也不同。

### SolidFlex 2000/PowerFlex 2000

适合已安装或计划安装光伏的用户，系统可直接接入光伏组件，实现：

- 光伏直充电池
- 光伏优先供电
- 晚上或用电高峰时用自己存的电，降低电费

下面是一个典型的工作流程示意：
1. 白天有阳光时：光伏组件发电，电能优先供给家庭正在使用的电器。
2. 电用不完时：多余的电能会自动存入电池模块中。
3. 夜间或用电高峰时：当光伏发电不足，系统会从电池中取电，为家庭负载供电。

系统会根据你的设置和当前用电情况自动运行，你只需在 App 中查看即可。

### SolidFlex 2000 ECO/PowerFlex 2000 ECO

ECO 系列不包含光伏直流输入接口，无法直接接入光伏组件，其主要特点是：

- 通过家庭插座与电网连接
- 在电价低的时候给电池充电
- 在电价高或需要时，用电池里的电供电  

**适合以下场景：**

- 尚未安装光伏，但希望先使用储能设备
- 所在地区有分时电价，希望降低用电成本
- 作为便携式或备用电源使用

下面是一个典型的工作流程示意：
- 低电价时段：系统从电网取电，将电能存入电池
- 高电价时段：家庭用电优先使用电池中的电
- 电池电量不足或达到设定阈值时：系统停止放电，用电切回电网。等到下一次低电价时段再充电  

同样，全程无需手动操作，App 中可随时查看运行情况。

## 系统组成

为了实现上述功能，整个储能系统由以下几个核心部分构成。

:::tip
不同型号在接口配置上可能略有差异，具体以所购型号为准。
:::

### 1. 储能主机

储能主机是整个系统的核心控制单元，负责“决定电该怎么用”。

它主要承担以下任务：

- 将光伏或电池产生的直流电转换为家庭可用的交流电
- 决定电能是直接使用、存入电池，还是从电池释放
- 实时监测系统运行状态，确保安全、稳定运行
- 根据设定执行功率限制、并网控制等策略
- 通过 Wi-Fi 与 App 通信，接收设置并上传运行数据

### 2. 电池模块

电池模块的作用很简单：存电，用电。

- 用于存储光伏多余电能
- 在夜间、阴天或用电高峰时释放电能
- 模块化设计，可根据需求逐步扩展容量
- 可直接与储能主机堆叠安装，结构紧凑、安装简便

如果你的用电需求增加，可以通过增加电池模块来扩展系统容量。

### 3. 光伏组件（可选）

光伏组件用于将太阳能转化为电能，是系统的主要能源来源之一。

- 通常安装在阳台、露台或其他光照条件良好的位置
- 白天优先为家庭负载供电，多余电能可存入电池
- 有助于降低对电网电力的依赖，减少电费支出

:::info
- SolidFlex 2000/PowerFlex 2000 ECO 系列型号无光伏接口，不支持直连光伏组件
- 即使暂时没有安装光伏，所有型号均可作为纯储能设备使用
:::

### 4. INDEVOLT App

INDEVOLT App 是你与储能系统之间的主要交互入口。

通过 App，你可以：

- 实时查看发电量、用电功率、电池电量等数据
- 设置用能策略，如自用优先、充放电计划
- 管理设备、进行固件升级
- 随时了解系统运行状态

### 5. 配件与辅助组件

为确保系统稳定、安全运行，系统还包含必要的配件，例如：

- 电源线缆，用于连接家庭插座（功率受系统设计及当地并网法规限制）
- 安装支架与固定组件，用于设备的稳固安装

所有配件均按照系统设计要求配置，以确保兼容性和使用安全。
