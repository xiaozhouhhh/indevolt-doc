# 开机与关机

本文档介绍如何通过设备按键，启动或关闭 Indevolt SolidFlex / PowerFlex 2000 储能设备。


## 开机操作

在以下情况下，你可能需要对设备进行开机操作：
- 设备刚完成安装或重新接线后
- 系统被手动关机，需要重新启动
- 长时间断电后恢复供电

**操作步骤：**
 
1. 确认设备当前为关机状态。
2. 长按电源按键约 2 秒。
3. 松开按键，设备开始启动，按键绿灯亮。
4. 蓝色 LED 灯条会显示从中间向两侧扩散的动画。
5. 当开机完成后，LED 灯条将根据当前系统状态显示[对应指示](../overview/appearance-interface.md#led)。


## 关机操作

在以下情况下，建议先将设备关机：
- 需要进行设备维护、清洁或接线操作
- 计划长时间不使用系统
- 需要安全断开外部电源

**操作步骤：**

1. 确认设备当前处于**开机状态**。
2. **断开光伏输入连接，并等待 20 秒以上**，确保系统停止发电输入。
2. 长按电源按键约 2 秒。
3. 蓝色 LED 灯条会显示从两侧向中间汇聚的动画。
4. 松开按键，系统开始执行关机流程。
5. 当按键灯完全熄灭时，表示设备已成功关机。


:::info

- 开机或关机过程中，请勿连续或反复按压电源按键。
- 关机后如需进行维护或接线操作，请等待设备完全停止运行。
- 若设备长时间不使用，建议关机并断开外部电源。
:::
