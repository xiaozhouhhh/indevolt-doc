# 查看状态与数据

你可以随时在INDEVOLT App查看设备当前运行情况，以及家庭的发电、储能和用电统计数据。

- **查看设备运行状态**  
  在App添加设备后，点击设备页面对应的设备卡片进入[设备详情](../../app/device.md#2-设备详情)，查看联网状态、设备运行模式、当前功率等


- **查看发电与用电数据**  
    在 App 底部导航栏中点击[统计](../../app/report.md)进入家庭能源数据总览页面，查看家庭日 / 周 / 月 / 年用电趋势、光伏发电量、节省的电费估算等。
