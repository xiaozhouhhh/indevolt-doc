# 安全须知

:::info
在使用本产品前，请仔细阅读本手册中的所有说明。请勿进行本手册未说明的任何改动或设置。因未按说明操作而导致的人身伤害、数据丢失或设备损坏，均不在保修范围内。
:::

- 为降低受伤风险，当本产品在儿童附近使用时，必须进行严格看护。  
- 请勿将手指或手伸入产品内部。  
- 使用非产品制造商推荐或销售的配件，可能会导致火灾、触电或人身伤害风险。  
- 为降低电源插头和电源线损坏的风险，断开产品连接时请拉动插头，而非电源线。  
- 请勿超出产品额定输出功率使用。超过额定值的过载可能引发火灾或人身伤害。  
- 请勿使用已损坏或被改装的产品或任何附件。损坏或改装过的电池可能出现不可预测的行为，导致火灾、爆炸或人身伤害。  
- 若电源线、插头或输出线缆损坏，请勿使用本产品。  
- 请勿自行拆解产品。如需维修或保养，请交由具备资质的维修人员处理。不正确的重新组装可能引发火灾或触电风险。  
- 若并网功率大于 800W，请务必安排具备资质的电工进行安装，以避免安全风险。  
- 请勿将产品暴露于火源或高温环境中。暴露在火焰中或高于 70°C 的温度可能导致爆炸。  
- 为降低触电风险，在按照说明进行任何维护操作前，请先将电源设备从插座中拔出。  
- 请由具备资质的维修人员进行维修，并仅使用完全相同的替换零部件，以确保产品的安全性能。  
- 在为设备充电时，产品表面可能会发热。这属于正常工作状态，无需担心。  
- 为内置电池充电时，请在通风良好的环境中进行，并确保通风口不被遮挡。  
- 请勿使用有害化学品或清洁剂清洁本产品。  
- 不当使用、跌落或施加过大的外力可能导致产品损坏。  
- 在处置二次电池或电池组时，应将不同电化学体系的电池或电池组分开处理。  
- 请勿在阳光直射的环境中长时间使用或存放本产品，例如车内、货厢或其他高温环境，否则可能导致产品性能下降、损坏或发热。  
- 电池的维修或维护应由熟悉电池特性及相关安全注意事项的人员进行或在其监督下进行。  
- 请勿在强静电或强磁场环境中使用本产品。  
- 请勿将本产品暴露在易燃、易爆气体或烟雾环境中。  
- 请勿站立或踩踏在产品上。  
- 请勿将产品浸入水中。如产品不慎落入水中，请将其放置在安全、通风的地方，远离火源，直至完全干燥后再使用。  
- 使用本产品时，请确保所连接的设备具备可靠的接地连接，以防止电气安全隐患。
