# 用电效率说明

Indevolt 微型储能可以把光伏发电或市电存入电池，并在家庭需要时释放使用。效率就是指“存进去的电，有多少最终能被你用上”。不同的使用场景会影响效率，下面为三种典型情况的说明。


## 1. 光伏电先存入电池再使用（PV → BAT → AC）

**使用场景：**

白天光伏发电产生的电，如果家庭用不完，会先存入电池，晚上或光伏不足时再用电池供家庭使用。

**损耗说明**

- 电池充电损耗（受光照波动影响）  
- 电池放电损耗  
- 逆变器转换损耗  

**典型效率示例：** 

光伏 (800W) → 电池 → 交流输出 (800W)


| 型号           | 效率   |
| -------------- | ------ |
| PowerFlex 2000 | ≥78.8% |
| SolidFlex 2000 | ≥78.8% |


## 2. 市电先存入电池再使用（AC → BAT → AC）

**使用场景：**
电网电价低时把市电存入电池，高价时再用，或者当作备用电源。

**损耗说明：**  

- 电池充放电损耗  
- 逆变器转换损耗  

**典型效率示例：**  

市电 (800W) → 电池 → 交流输出(800W)

| 型号           | 效率   |
| -------------- | ------ |
| PowerFlex 2000 | ≥80.2% |
| SolidFlex 2000 | ≥80.2% |


## 3. 光伏电直接供家庭使用（PV → AC）

**使用场景：**
白天家庭用电刚好匹配光伏发电量，电能不经过电池直接使用。

**损耗说明**

- 不经过电池充放电，能量损耗最少  
- 只有逆变器转换和 MPPT 跟踪会有少量损耗  


**典型效率示例：**  

光伏 → 逆变器 → 交流输出（满载）

| 型号           | 效率 |
| -------------- | ---- |
| PowerFlex 2000 | ≥93% |
| SolidFlex 2000 | ≥93% |
