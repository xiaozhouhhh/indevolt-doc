# 深度待机与普通待机说明

## 1. 待机状态介绍

当储能设备暂时没有充电或供电需求时，系统会进入待机状态，以降低设备运行功耗。待机状态下，设备仍会持续监测电池状态，并在检测到充电或供电需求时自动恢复运行。

根据节能程度不同，待机状态可分为：

- **普通待机**：适用于短时间无充放电需求，设备保持较快响应能力。
- **深度待机**：适用于长时间无充放电需求，通过关闭部分非必要功能进一步降低功耗。

深度待机是普通待机基础上的进一步节能状态，并不是设备关机。


## 2. 深度待机与普通待机区别

| 对比项      | 普通待机               | 深度待机           |
| ----------- | ---------------------- | ------------------ |
| 使用场景    | 短时间无充放电需求     | 长时间无充放电需求 |
| 功耗        | 较低                   | 更低               |
| 系统状态    | 逆变器始终处于待命状态 | 逆变器会进入休眠   |
| 恢复速度    | 更快                   | 相对较慢           |
| EPS切换时间 | 约100ms                | 约1~2s             |

👉 查看详细功耗说明：[待机功耗](../technical-note/standby-power.md#不同运行状态下的功耗说明)

> 注：由于深度待机需要恢复部分运行功能，因此退出深度待机后，EPS（应急供电）的切换时间会比普通待机稍长。


## 3. 深度待机介绍

### 3.1 什么是深度待机

深度待机是一种低功耗运行状态。当设备长时间没有检测到充电需求或输出需求时，系统会自动进入深度待机，以减少设备自身耗电。

进入深度待机后：

- 电池保护功能仍会持续运行；
- 设备会持续检测外部输入和用电需求；
- 检测到需求后会自动恢复正常工作。

### 3.2 为什么进入深度待机

储能设备即使没有充电或供电，也需要消耗少量电量维持系统运行。

例如：

- 夜间没有光伏发电且没有负载使用；
- 设备长时间闲置。

深度待机可以减少设备在这些低使用场景下的待机功耗，从而降低电池电量消耗。


### 3.3 如何设置深度待机

深度待机策略支持配置生效时间：

- 可设置开始时间和结束时间；
- 支持跨天设置；
- 默认配置为全天生效（00:00 - 23:59）。

设置方法：
1. 在设备设置页面选择 **深度待机**；
2. 设置深度待机生效时间。

### 3.4 如何退出深度待机

设备检测到以下情况时，会自动退出深度待机：

- 有充电需求；
- 有供电需求；
- 检测到外部电源输入（如光伏或交流输入）；
- 通过设备按键操作。

退出深度待机后，设备将恢复正常工作模式。
