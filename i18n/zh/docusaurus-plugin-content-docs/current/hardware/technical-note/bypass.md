# 旁路口使用说明

旁路口（Backup）是一个**可自动切换方向的接口**：

- 接入光伏微逆 → 作为**输入口（发电）**
  <img src={require("./img/microinverter.png").default} width="300"/> 

- 接入电器负载 → 作为**输出口（供电）**
  <img src={require("./img/eps.png").default} width="300"/> 

👉 系统会自动识别当前连接类型，并切换运行模式。


:::info 固件要求
EMS ≥ 1.01.05 支持旁路口连接负载或微逆。
:::

## 接入设备识别

旁路口会根据功率自动判断接入设备类型：

| 条件                           | 识别为 | 运行模式 |
| ------------------------------ | ------ | -------- |
| **输入**功率 ≥ 30W，持续 5 秒  | 微逆   | 微逆模式 |
| **输出**功率 ≥ 30W，持续 20 秒 | 负载   | EPS 模式 |


## 设备基本行为

- 当旁路口功率为 0 时，设备会保持“上一次运行模式”
- 设备重启后，默认进入 **EPS 运行模式**（需断开 PV 和 AC 后再重启）


## 旁路口断开/恢复逻辑


**系统最低配置**
- 1 x SolidFlex 2000 + 2 x SFA1800
- 1 x PowerFlex 2000 + 1 x PFA2000 

**并网**


|   对象   | 正常范围 | 触发断开条件    | 恢复时间 |
| ---- | ------------ | -------------- | -------------- |
| 微逆 | 功率 ≤ 2400W      | - 功率 > 2400W，持续时间 > 30s<br />- 或超出最大馈网功率 10% 且持续时间 > 30s（详见[用能模式运行逻辑](#用能模式运行逻辑)） | 15min          |
| 负载 | 功率 ≤ 3600W      | - 电流 > 16A，持续时间 > 5s <br />- 或SOC < 5%   | 15min     |


**离网**

|  对象 | 工作范围 | 触发断开/不响应条件          | 恢复条件 |
| ---- | ------------ | --------------- | -------------- |
| 微逆 | 最低固件版本：EMS V1.01.05<br />范围：功率 30W ~ 1200W（系统只消纳1200W）；SOC ≥ 3%     | SOC < 3% 系统不响应微逆（不会断开） | SOC ≥ 3% 系统响应微逆         |
| 负载 |功率范围：<br />- ≤ 2400W＋2%，稳定运行<br />- 2448W ~ 2640W，支持 30s 过载<br />- 2640W ~ 3000W，支持 15s 过载<br />- > 3000W，支持 100ms 过载 | 过载  | 15min |


## 用能模式运行逻辑

### 保护原则

所有用能模式都受以下规则控制，优先级最高，且全局通用：

- 微逆功率 ＞ 馈网限制 → 优先断开旁路（断微逆）
- 微逆功率 ＜ 馈网限制 → 优先降低 PV 输出

### 自用优先

- 光伏（微逆+PV）优先供给负载
- 剩余电量再存入电池
- 电池充满后：
   - 在馈网限制内 → 并网
   - 超出馈网限制部分 → 判断微逆功率是否大于馈网功率限制 → 执行断开旁路或降 PV 逻辑

### 自定义充放计划 （仅PV）

**充电**

- 光伏（微逆+PV）优先充电池
- 电池充满后：判断微逆功率是否大于馈网功率限制 → 执行断开旁路或降 PV 逻辑

**放电**

- 光伏（微逆+PV）优先供给负载
- 剩余电量再存入电池
- 电池充满后：
   - 在馈网限制内 → 并网
   - 超出馈网限制部分 → 判断微逆功率是否大于馈网功率限制 → 执行断开旁路或降 PV 逻辑

### 自定义充放计划 （PV+AC）

**充电**

- 光伏（微逆+PV）+ AC 优先充电池
- 电池充满后：AC 停止充电 → 判断微逆功率是否大于馈网功率限制 → 执行断开旁路或降 PV 逻辑

**放电**
- 光伏（微逆+PV）+ AC 供给负载，供给优先级：微逆+PV → BAT → AC
- 微逆+PV 剩余电量再存入电池
- 电池充满后：
   - 在馈网限制内 → 并网
   - 超出馈网限制部分 → 判断微逆功率是否大于馈网功率限制 → 执行断开旁路或降 PV 逻辑

**实时控制**

**充电**
- 光伏（微逆+PV）+ AC 优先充电池
- 电池充满后：微逆+PV 馈网
- 超过3600W → 判断微逆功率是否大于馈网功率限制 → 执行断开旁路或降 PV 逻辑

**放电**
- 光伏（微逆+PV）+ AC 供给负载，供给优先级：微逆+PV → BAT → AC
- 微逆+PV 剩余电量再存入电池
- 电池充满后：
   - 充满后（微逆+PV）馈网
   - 超过3600W → 判断微逆功率是否大于馈网功率限制 → 执行断开旁路或降 PV 逻辑

**静止**
- 光伏（微逆+PV）优先充电池
   - 充满后（微逆+PV）馈网
   - 超过3600W → 判断微逆功率是否大于馈网功率限制→执行断开旁路或降PV逻辑

**离网补充说明**

如果离网的休眠设置 → 待机关机倒计时到达后，设备会进入**休眠**，旁路口会**自动关闭**，可通过以下方式重新开启：
1. 点按电源键 
2. 根据网络情况选择对应控制方式开启
