# 待机功耗说明

在日常使用中，您可能会发现这样的情况：设备没有在充电或放电，但电池电量在缓慢下降。其实，大多数情况下，这是系统的正常运行特性。

**为什么设备在“没有工作”时也会耗电？**

储能设备并不是只有在充放电时才工作。即使没有对外供电，系统内部仍然需要维持基本运行，如控制系统需要随时待命，电池管理系统（BMS）需要持续监测电池状态，通信模块需要保持在线等。

只要设备处于通电状态，这些模块就会消耗少量电能。这部分能耗可以理解为设备的“维持运行成本”。因此，在待机状态下看到少量功耗，是正常现象。

## 影响待机功耗的主要因素

系统待机功耗并不是单一固定值，而会随着系统配置和运行条件变化而变化。影响待机功耗的主要因素包括：

### 1. 是否连接电网（并网 / 离网）

**并网状态**

设备连接电网时，部分功能可以依赖电网运行，因此系统功耗通常较低。

**离网状态**

当设备处于离网状态时，系统需要自行维持供电能力，因此会保持更多模块运行，待机功耗通常更高。

简单理解：

- 并网：系统可以借助电网，功耗较低
- 离网：系统需要自己维持供电能力，功耗更高


### 2. 是否开启旁路

旁路开启时，若电网可用，负载可由电网和主机电池直接供电。这种情况下，系统主要维持控制、检测和旁路相关回路运行，因此功耗通常低于离网逆变维持状态，但高于深度待机状态。

当旁路关闭且系统离网时，设备自动进入深度待机，从而降低功耗。

简单理解：

- 旁路开启：电网直供负载，系统在旁边“值班”
- 旁路关闭：系统更可能需要保持逆变路径待命或运行


### 3. 是否开启深度待机

深度待机功能开启后，系统处于低功耗模式。当系统没有充放电需求时，部分高功耗模块（例如逆变器）会进入休眠状态，仅保留基础监控和通信功能，从而降低功耗。

简单理解：

- 开启深度待机：设备进入省电模式
- 关闭深度待机：设备保持随时响应状态，功耗相对更高


### 4. 是否具备 AC 辅助电源

AC 辅助电源用于为设备内部控制系统提供电源。新版的PowerFlex / SolidFlex 2000设备配备 **AC 辅助电源**。

如果设备连接电网并具备 AC 辅源，部分内部模块可以由电网供电，从而减少电池消耗。

简单理解：

- 有 AC 辅源：部分待机功耗来自电网
- 没有 AC 辅源：这些功耗可能来自电池

## 不同运行状态下的功耗说明

根据是否连接电网以及系统工作方式，设备会有不同的运行状态。

:::note
- 以下内容仅适用于 PowerFlex / SolidFlex 2000 系列，不适用于 PowerFlex / SolidFlex 1200 和 3000 系列。
- 以下功耗为典型值，实际数值可能因环境温度、通信状态、电池数量等因素略有浮动。
- 总功耗 = 主机功耗（包含主机内部电池及控制系统）+ 电池包数量 × 电池包功耗。
:::

### 离网状态

离网状态下，系统无法依赖电网，需要自行维持供电能力，因此待机功耗通常高于并网状态。

**🔌旁路开启**

系统需要保持逆变器运行能力，以便随时为负载供电。**无论深度待机开关是否开启，功耗均相同。**

:::note
如果开启旁路后，离网的**休眠设置** - **待机关机倒计时**到达后，此时功耗与旁路关闭时相同。
:::


| 型号                    | 主机功耗（W） | 电池包功耗（W） |
| ----------------------- | -------- | ---------- |
| PowerFlex 2000          | 29       | 5          |
| SolidFlex 2000          | 29       | 5          |
| PowerFlex 2000（配备AC辅源） | 29       | 5          |
| SolidFlex 2000（配备AC辅源） | 29       | 5          |



**🔌旁路关闭**

在离网状态且旁路关闭时，设备会自动进入深度待机。
