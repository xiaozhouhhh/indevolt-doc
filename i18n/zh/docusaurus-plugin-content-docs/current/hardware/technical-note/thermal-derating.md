# 光伏/电池温度降额说明

设备在适宜温度范围内可以满功率运行；当温度过高或过低时，系统会自动降低功率，这种行为称为降额运行（Derating）。

降额是一种正常的保护机制，本质是让设备在“更安全的状态”下运行：一方面避免因过热或过冷导致故障，另一方面也有助于延长设备使用寿命。当温度恢复到安全范围后，系统会自动恢复额定功率，无需人工干预。

---

## 光伏过温降额

正常情况下，系统会尽可能提高发电量；但当温度升高时，持续满功率运行反而会带来额外压力。当设备内部的光伏转换模块（MPPT）温度过高时，系统会自动降低光伏功率以保护设备。


:::info
ECO 系列不包含光伏直流输入接口，无法直接接入光伏组件，不适用本节内容。
:::

**为什么会降额？**

光伏转换模块在进行功率转换（DC-DC）时会持续发热，并且功率越大，发热越明显。同时，外部环境也会进一步加剧温升，例如：

- 夏季中午环境温度较高  
- 安装环境通风不良  

当温度持续升高，如果仍保持满功率运行，可能导致器件过热、效率下降，甚至损坏。因此系统会主动降低输出功率，以换取更安全的运行状态。

**会有什么表现？**

在实际使用中，光伏降额表现为：

- 光伏发电功率达不到理论峰值  
- 中午时段发电曲线出现“被压低”的情况  

这些现象均属于正常保护行为。

**降额说明**

当设备内部光伏转换模块（MPPT）温度升高时，系统会按阶梯限制光伏最大功率；温度下降后再自动恢复：

- 当温度高于 **93℃**，最大功率降到额定功率的 **80%**
  - 例如：总功率降至 `2400W x 80% = 1920W`；单路 PV 降至 `600W x 80% = 480W`
  - 当温度低于 **75℃** 后恢复额定功率

- 当温度高于 **96℃**，最大功率降到额定功率的 **60%**
  - 例如：总功率降至 `2400W x 60% = 1440W`；单路 PV 降至 `600W x 60% = 360W`
  - 当温度低于 **80℃** 后恢复额定功率

:::note
当前 App 暂不支持显示光伏转换模块（MPPT）温度。
:::

## 电池温度降额

相比光伏转换模块，电池对温度更加敏感，因此其保护策略也更加严格。无论是高温还是低温，都可能触发降额。

**为什么会降额？**

电池内部是化学反应过程，对温度变化非常敏感：

- 温度过高：加速老化，甚至存在安全风险  
- 温度过低：化学反应变慢，充电可能造成损伤  

因此系统需要通过限制充放电能力，来确保电池始终工作在安全区间。


**会有什么表现？**

- 充电速度变慢，甚至无法充电  
- 放电功率下降  
- 无法达到额定充放电功率  

**降额说明**

电池最大充放电功率会随温度变化动态调整：
- **高温时**：系统会降低充电功率，同时限制放电能力；如果温度继续上升，最终停止充放电，以避免过热风险。
- **低温时**：系统主要限制充电，因为低温充电可能对锂电池造成不可逆损伤；同时放电能力也会下降。因此在冬季，更容易出现“放电正常，但无法充电”的情况。
