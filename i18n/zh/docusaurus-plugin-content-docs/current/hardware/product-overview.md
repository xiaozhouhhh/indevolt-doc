# 产品概述

Indevolt微型储能系统是一种**即插即用的家庭微型储能解决方案**，通常与光伏组件配合使用。  

系统可将白天产生的太阳能电力进行存储，并在家庭需要时释放，为家庭负载供电，从而提高光伏发电的利用效率，降低对电网的依赖。

该系统无需复杂施工或改造原有配电结构，适合希望以低门槛方式使用清洁能源的家庭用户。

## 典型应用场景

- ✅公寓或住宅阳台空间有限的家庭。
- ✅无法安装大型屋顶光伏系统的用户。
- ✅欧盟并网功率受限地区（如600W / 800W / 2000W）。
- ✅希望提高光伏发电自用比例、减少电网购电的家庭。
- ✅在分时电价或高峰电价环境下，通过价格策略降低用电成本。

## 产品优势

- 🔌 即插即用，无需复杂施工。
- 🖥️ 支持本地运行，不依赖云端。
- 🔋 模块化设计，支持按需扩展储能容量。
- 🛡️ 符合欧盟安全与并网标准。
- 📱 通过 App 实时查看状态并调整用能策略。
- 🔗 提供本地 API，便于接入第三方平台或智能家居系统，实现高级自动化控制。
