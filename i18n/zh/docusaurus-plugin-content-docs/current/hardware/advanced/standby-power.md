# 待机功耗说明

在日常使用中，您可能会发现这样的情况：设备没有在充电或放电，但电池电量在缓慢下降。其实，大多数情况下，这是系统的正常运行特性。

**为什么设备在“没有工作”时也会耗电？**

储能设备并不是只有在充放电时才工作。即使没有对外供电，系统内部仍然需要维持基本运行，如控制系统需要随时待命，电池管理系统（BMS）需要持续监测电池状态，通信模块需要保持在线等。

只要设备处于通电状态，这些模块就会消耗少量电能。这部分能耗可以理解为设备的“维持运行成本”。因此，在待机状态下看到少量功耗，是正常现象。


## 不同运行状态下的功耗说明

根据是否连接电网以及系统工作方式，设备会有不同的运行状态。

:::note
- 以下内容仅适用于 PowerFlex / SolidFlex 2000 系列，不适用于 PowerFlex / SolidFlex 1200 和 3000 系列。
- 以下功耗为典型值，实际数值可能因环境温度、通信状态、电池数量等因素略有浮动。
- 总功耗 = 主机功耗 + 电池包数量 × 电池包功耗。
:::


### 1. 深度待机模式（AC连接）

当设备连接市电，同时没有进行充电或放电，且旁路关闭时，系统会处于待命状态。

典型功耗（单位：W）

| 型号          | 主机功耗 | 电池包功耗 |
| ------------- | -------- | ---------- |
| PowerFlex 2000 | 7        | 2          |
| SolidFlex 2000 | 7        | 2          |

此时系统功耗最低，这种状态适合长期保持设备在线，是日常使用中最常见、也是最稳定的运行方式。

### 2. 离网运行模式（AC断开）

当市电断开或设备主动进入离网模式时，系统是否持续运行主要取决于旁路开关状态。当旁路开启时，系统需要持续维持输出能力，以便随时为负载供电。此时设备需要维持逆变器运行，因此功耗会明显高于深度待机状态。

典型功耗（单位：W）

| 型号          | 主机功耗 | 电池包功耗 |
| ------------- | -------- | ---------- |
| PowerFlex 2000 | 29        | 5         |
| SolidFlex 2000 | 29        | 5         |


此时所有能耗都来自电池。在没有光伏补充的情况下，电池电量会随着时间逐渐下降。因此，离网模式更适合停电备用或短时间离网使用。

如果离网状态下旁路关闭，系统会自动进入深度待机状态，以降低功耗。可参考 [深度待机模式](#1-深度待机模式ac连接)。


### 3. 旁路模式（AC连接）

在旁路模式下，旁路开关打开，负载由市电和电池共同供电。设备自身功耗低于离网运行模式，但仍高于深度待机模式。

典型功耗（单位：W）

| 型号          | 主机功耗 | 电池包功耗 |
| ------------- | -------- | ---------- |
| PowerFlex 2000 | 20        | 0         |
| SolidFlex 2000 | 20        | 0         |
