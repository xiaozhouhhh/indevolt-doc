# 三相电表 CT 多位置监测

在三相系统中，L1、L2、L3 三条相线通常各安装一个 CT，用于测量整个三相系统的功率与电量数据。

但三个 CT 并不一定需要分别安装在三条相线上，也可以根据实际需求分别夹在光伏、电网、负载或电池等不同回路中，从而同时获取系统不同位置的运行数据。

---

## 1. CT 安装位置

CT 安装位置不同，得到的数据也不同。

| CT 安装位置 | 对应数据                          | 典型用途         |
| ----------- | --------------------------------- | ---------------- |
| 光伏侧      | 光伏发电功率                      | 查看发电情况     |
| 电网侧      | 购电 / 售电功率（取决于电流方向） | 防逆流、并网监测 |
| 负载侧      | 用电功率                          | 统计设备耗电     |
| 电池侧      | 充电 / 放电功率                   | 储能状态监控     |

不同位置的 CT 在同一时刻会显示不同数据，这些数据共同反映整个系统的运行情况。

---

## 2. CT 安装方向

CT 外壳上的箭头用于指示电流参考方向。如果安装方向错误，不会损坏设备，但会导致功率正负方向相反。

例如，实际从电网购电时可能显示为向电网送电，电池充电可能显示为放电。

如出现数据方向异常，请检查：
- CT 是否安装方向相反
- App 中互感器方向设置是否正确

:::note
如果 CT 物理安装方向不方便调整，也可以在 App 中修改 CT 方向进行校正。
:::

---

## 3. App 配置

### 3.1 设置 CT 安装位置

1. 进入电表设置页面，选择**电表配置** > **单相系统**。
2. 为三个 CT 分别选择实际安装位置（光伏、电网、负载或电池）。
3. 根据 CT 实际箭头方向，设置对应的 CT 方向。可根据图示判断方向是否符合实际安装情况。

配置完成后，可在电表页面分别查看三个 CT 对应位置的运行数据，例如功率、电量、电流和电压等信息。

## 4. 如何判断是否配置正确

配置完成后，可通过实际运行情况进行验证：
- 光伏发电时，光伏功率应为正值
- 从电网用电时，应显示购电状态
- 电池充电时，应显示充电功率
- 负载开启时，对应用电功率应上升

如出现异常数据或方向相反，可依次检查：
- CT 是否安装在正确位置
- CT 安装方向是否正确
- App 中 CT 配置是否与实际一致
- CT 是否夹紧并完全闭合
