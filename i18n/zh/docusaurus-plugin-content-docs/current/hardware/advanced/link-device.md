# 关联设备

通过将微储系统与家庭中的其他设备（如智能插座、智能电表或读表器）进行关联，系统可以在局域网内安全共享用电数据，从而更准确地判断用电需求，并自动调整充放电功率。

## 为什么需要关联其他设备

如果微储没有关联任何其他设备，它在运行时看不见负载变化，只能按固定规则工作：

- 不知道家里当前用了多少电，无法根据实时用电变化自动调整功率  
- 只能按固定功率或时间计划充放电   

在实际使用中，可能会出现：

- **放电过多** → 电能反向送入电网 
- **放电不足** → 家庭仍需从电网取电，电池没能充分发挥作用

:::tip
因此，关联其他设备的核心作用就是**为微储“提供用电反馈”，让系统能实时感知负载变化，动态调整充放电行为。**
:::

## 可以关联哪些设备

根据您的使用场景不同，微储系统可关联以下类型的设备：

### 智能插座

适合需要精准控制**某一路或某几台设备**用电的场景。

**可以做什么？**

- 实时获取插座上设备的用电功率  
- 微储会“跟着这路负载”放电，精准匹配功率  
- 仅为该插座上的设备补电，不影响家中其他回路  

### 智能电表

适合希望实现**整屋用电管理**的用户。

**可以做什么？**

- 实时获取家庭总用电功率  
- 判断当前是从电网取电，还是向电网馈电  
- 微储根据电表数据自动调节功率，实现 **零馈网** 或最大化自用


### 读表器

当电表本身无法直接联网时的替代方案。

**工作方式：**

- 微储不直接连接电表  
- 通过读表器读取电表数据  
- 根据读取的用电信息间接调节微储功率  

## 如何关联设备

:::info 准备
在开始关联前，请先确认以下条件已满足：

- 微储设备与要关联的设备（如智能插座、电表、读表器）**已添加到同一个家庭**
- 所有关联设备 **均已开机并正常联网**

👉 如尚未添加设备，请先参考：[添加设备](../../app/add-device.md)
:::

1. 在 INDEVOLT App **设备** 页面点击微储设备卡片，进入 **设备详情页**
2. 在页面下方点击 **+ 新增子设备**
3. 在「可关联设备」列表中，选择需要关联的设备
4. 点击 **关联**，等待系统完成配置
