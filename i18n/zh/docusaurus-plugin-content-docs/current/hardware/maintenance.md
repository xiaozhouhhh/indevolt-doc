# 存储与维护


1. 存储时请按包装箱标识放置电池，切勿倒置或侧放。
2. 轻拿轻放电池，避免损坏。
3. 存储环境要求如下：
   - 环境温度：-20°C至60°C；推荐存储温度：20°C~30°C
   - 相对湿度：5%至80%
   - 置于干燥、清洁且通风良好的场所
   - 远离腐蚀性有机溶剂和气体
   - 避免阳光直射
   - 距离热源至少2米
4. 存储中的电池必须断开外部设备连接，指示灯应处于熄灭状态。
5. 请勿将电池投入火中，可能引发爆炸。
6. 禁止拆解或破坏电池，泄漏的电解液对皮肤和眼睛有害，可能具有毒性。
7. 清洁时请使用棉布和水，禁用钢丝球等硬质材料。
8. 长期存储时，每3个月进行一次充放电（放电至20%电量后充至80%）。
9. 电池维护需由具备电池专业知识的人员操作或监督。
10. 若电池出现变形、破损或漏液，无论存储时间长短均需直接废弃。
