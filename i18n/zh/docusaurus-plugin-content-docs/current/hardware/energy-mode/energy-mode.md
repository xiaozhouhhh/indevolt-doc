# 能量管理

能量管理用于控制储能系统在不同时间、不同用电条件下的充放电行为。  
通过合理选择用能模式，你可以决定**电能从哪里来、在什么时候使用，以及如何分配**，从而提升能源利用效率并降低用电成本。


## 能量管理能帮你做什么？

通过能量管理，你可以：

- 优先使用自家光伏发电，减少从电网购电  
- 在合适的时间为电池充电或放电  
- 根据用电习惯或电价变化优化用能成本  
- 在需要时手动或自动控制系统运行行为  

系统提供多种用能模式，以满足不同家庭和使用场景的需求。
