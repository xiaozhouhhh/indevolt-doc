# 价格策略


价格策略是一种智能用能模式。系统会根据你设定的电价阈值，自动判断什么时候该充电、什么时候该放电，在尽量不影响用电体验的前提下，帮助你降低电费支出。


## 适合哪些用户

- 所在地区采用 **分时电价 / 动态电价 / 实时电价**
- 希望利用“低价存电、高价用电”来节省电费
- 不想每天手动安排充放电计划
- 希望系统能根据价格变化自动决策

## 系统如何工作

在价格策略模式下，系统会根据当前电价，自动在以下三种状态之间切换：

**充电（低电价时）**

设备的实际充电功率由逆变器的最大输入功率决定。

- 系统优先使用光伏电量为电池充电；
- 当光伏电量不足时，系统将自动从电网补电；
- 当电池充满后，充电任务自动结束。

**放电（高电价或高用电需求时）**

实际放电功率由当前负载来源（电表/插座/默认负载）决定。

- 设备优先使用光伏供电；
- 光伏不足时，电池自动补充供电；
- 当电池电量达到应急电量值时，将自动停止放电。

**静止（不在策略充放时段）**

在静止状态下，电池不会主动充电或者放电

- 光伏发电优先供给家中用电设备；
- 当光伏功率大于负载且电池SOC未达100%时, 多余电量将自动回充至电池；
- 电池充满后，光伏功率会自动被限制。
