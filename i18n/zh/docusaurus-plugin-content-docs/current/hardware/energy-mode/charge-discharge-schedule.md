# 自定义充放计划

自定义充放计划模式允许你按照一天中的不同时间段，明确指定系统在什么时间充电、什么时间放电，或保持待机状态。

一旦设置完成，系统将严格按照你的计划自动运行，无需每天手动调整。

## 适合哪些用户

如果你符合以下任意情况，自定义充放计划会非常适合你：
- 所在地区有 分时电价 / 峰谷电价
- 白天和夜间用电差异明显
- 作息时间和用电习惯比较固定


## 系统如何工作

在该模式下，你可以将一天划分为最多 24 个时间段，并为每个时间段指定运行方式：

- 充电：系统按设定功率给电池充电
- 放电：系统按负载情况或设定功率供电
- 静止：系统不主动充放电

**用电优先级**

- 家庭用电 **优先使用光伏电能**
- 光伏不足时，由 **电池放电补充**
- 光伏和电池都不足时，才从 **电网取电**

## 负载选择


| 负载选择             | 放电时间段 | 充电时间段 |
| -------------------- | ---------- | ---------- |
| 智能插座             | 系统根据智能插座上连接的实际负载功率进行放电。<br/>可设置 <u>额外功率补偿</u> 用于微调输出：实际放电功率 = 插座负载功率 + 额外功率补偿 | 系统按照计划中设定的功率进行充电 |
| 智能电表 (电表需安装在 电网进线侧)  |  系统会以“电表功率为 0”为目标进行调节：<br/>- 电表功率为正 → 系统放电补偿用电 <br/>- 电表功率为负 → 系统充电吸收多余电能  | 系统按照计划中设定的功率充电 |
| 关键负载             | 系统仅为旁路口连接的设备供电，不向电网放电  | 系统按照计划中设定的功率充电|
| 自定义设置（负载曲线 | 系统按照你设定的负载曲线功率放电 | 系统按照计划中设定的功率充电 |


:::info
当负载设置为智能插座或智能电表时，如果智能插座或智能电表离线，系统会自动切换负载曲线功率放电。
:::

:::info
为避免超载或异常，系统会自动取最小安全值执行：
- 放电时：最大交流输出功率、负载曲线、负载功率 → 取最小值
- 充电时：逆变器最大输入功率、充电功率 → 取最小值
:::

## 如何设置自定义充放计划

**操作步骤：**

1. 确认您已在 **INDEVOLT App** 添加设备至您的家庭。
2. 点击底部 **设备** 标签，选择对应设备卡片进入设备详情页
3. 点击用能模式区域，该区域根据您所选的模式动态显示，进入模式选择页面
4. 选择 **自定义充放计划**，点击 **去设置**
5. 点击 **＋ 添加计划**：
   - 设置开始时间 / 结束时间
   - 选择充电 / 放电
   - 设置对应功率
6. 点击 **保存**，设置立即生效

## 推荐设备参数设置

在设备详情页，点击右上角 **⚙ 设置** 图标进入设置页面。

<img src={require("./img/device_info.png").default} width="240"/>
<img src={require("./img/device_setting.png").default} width="240"/>

为了获得更稳定的使用体验，建议如下设置：

| 设置项                     | 建议配置                               |
| -------------------------- | -------------------------------------- |
| 最大交流输出功率（放电）   | 不低于家庭实际负载功率，并符合当地法规 |
| 逆变器最大输入功率（充电） | 不低于期望的充电功率                   |
| 最大馈网功率               | 建议设置为 `0`（开启防逆流）           |
| 旁路开关                   | 无特殊需求时保持关闭                   |
| 应急电量                   | 建议设置为 20%                         |
