# 自用优先

在**自用优先**模式下，Indevolt 微型储能系统会根据家庭用电情况，**实时调节充放电功率**，尽可能优先使用光伏发电，减少电网购电，并避免向电网反向馈电。

整个过程无需人工干预，系统会自动根据配置完成能量分配。


## 适合哪些用户

如果你符合以下情况，推荐使用自用优先模式：

- 已安装光伏组件，希望充分利用自发电能  
- 希望降低电网购电量和电费支出  
- 日常用电较为规律，不想频繁调整设置  
- 希望系统自动运行、操作尽量简单  


## 系统如何工作

在自用优先模式下，系统会根据发电和用电情况，自动进行能量分配。

**发电优先级**

- 光伏发电 **优先供给家庭负载**
- 多余的光伏电量 **用于给电池充电**
- 当电池充满后，剩余电量才可能馈入电网（取决于馈网设置）

**用电优先级**

- 家庭用电 **优先使用光伏电能**
- 光伏不足时，由 **电池放电补充**
- 光伏和电池都不足时，才从 **电网取电**


## 负载选择

为了让系统准确判断需要多少放电功率，你需要选择合适的负载类型：

**智能插座**

- 系统会根据智能插座上连接的实际负载功率进行放电
- 可设置额外功率补偿，用于微调输出功率
- 实际调度功率 = 插座负载功率 + 额外功率补偿
- 适合只为部分设备供电，或希望精确控制放电功率的用户

**智能电表**

- 电表需安装在 **电网进线侧**
- 系统会以“电表功率为 0”为目标进行调节：
  - 电表功率为正 → 系统放电补偿用电
  - 电表功率为负 → 系统充电吸收多余电能  
- 适合希望实现整屋零馈网、最大化自用的用户

**关键负载**

- 系统仅为旁路口连接的负载供电，不会向电网放电
- 适合只需保障关键设备的用户


**自定义设置（负载曲线）**

- 可将一天划分为最多 48 个时间段，每个时间段设置不同的输出功率

:::info
当负载设置为智能插座或智能电表时，如果智能插座或智能电表离线，系统会自动切换负载曲线功率放电。
:::

:::warning
如果负载曲线设置过高或不合理，可能会出现馈网现象。
:::


## 如何设置自用优先

**操作步骤：**

1. 确认您已在 **INDEVOLT App** 添加设备至您的家庭
2. 点击底部 **设备** 标签，选择对应设备卡片进入设备详情页
3. 点击用能模式区域，该区域根据您所选的模式动态显示，进入模式选择页面
4. 选择 **自用优先**，点击 **去设置**
5. 根据实际情况选择[负载类型](#负载选择)（智能插座 / 智能电表 / 关键负载 / 自定义设置）
6. 点击 **保存**，设置立即生效

## 推荐设备参数设置

在设备详情页，点击右上角 **⚙ 设置** 图标进入设置页面。

<img src={require("./img/device_info.png").default} width="240"/>
<img src={require("./img/device_setting.png").default} width="240"/>


为了获得更稳定的使用体验，建议如下设置：

| 设置项             | 建议配置                               |
| ------------------ | -------------------------------------- |
| 最大交流输出功率（放电）   | 不低于家庭实际负载功率，并符合当地法规 |
| 逆变器最大输入功率（充电） | 不低于期望的充电功率                   |
| 最大馈网功率       | 建议设置为 `0`（开启防逆流）           |
| 旁路开关           | 无特殊需求时保持关闭                   |
| 应急电量           | 建议设置为 20%                         |
