# 售后支持与保修

1. 有限保修  
我们向原始消费者及购买者承诺，本产品在下方第2条规定的保修期内正常使用情况下，不存在工艺和材料缺陷（第5条所列除外情形除外）。本保修声明构成全部保修责任范围。对于产品销售相关的其他责任，我们既不承担，也不授权任何他人代为承担。

2. 保修期限  
凭购买凭证完成产品注册后生效。涵盖正常使用下的制造缺陷；不包含耗材、人为损坏或未经授权的改装。具体条款依地区政策而定。

3. 维修方案  
在适用保修期内，对于因工艺或材料缺陷导致的故障产品，我们将自行选择并承担费用进行维修或更换。

4. 仅限原始消费者  
本服务仅限于原始消费者所购产品的保修权益，不可转让给后续所有者。

5. 免责条款  
本产品保修不适用于以下情况：
因误用、滥用、改装、事故或基于当时产品信息未经授权的非正常用途导致的任何产品损坏。
