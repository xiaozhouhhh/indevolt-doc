# 开机与关机

本文档介绍如何通过设备按键对Indevolt储能系统进行开机和关机操作。

## 开机操作

在以下情况下，你可能需要对设备进行开机操作：
- 设备刚完成安装或重新接线后
- 系统被手动关机，需要重新启动
- 长时间断电后恢复供电

**操作步骤：**
1. 确认设备当前为关机状态。
2. **长按电源按键约 2 秒**。
3. 松开按键，设备开始启动。
4. 蓝色 LED 指示灯 从中间向两侧逐渐点亮，表示系统正在启动。

当指示灯状态稳定后，说明系统已成功开机，并会根据当前设置自动开始工作，无需额外操作。

## 关机操作

在以下情况下，建议先将设备关机：
- 需要进行设备维护、清洁或接线操作
- 计划长时间不使用系统
- 需要安全断开外部电源

**操作步骤：**
1. 确认设备当前处于开机状态。
2. **长按电源按键约 2 秒**。
3. 松开按键，系统开始执行关机流程。
4. 蓝色 LED 指示灯 从两侧向中心汇聚，表示系统正在关机中。

动画结束后，设备即已完全关闭。


:::info

- 开机或关机过程中，请勿连续或反复按压电源按键。
- 关机后如需进行维护或接线操作，请等待设备完全停止运行。
- 若设备长时间不使用，建议关机并断开外部电源。
:::
