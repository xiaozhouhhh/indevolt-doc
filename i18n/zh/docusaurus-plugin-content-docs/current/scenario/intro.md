# 单相系统（≤800W）+ 阳台光伏说明

## 一、这个方案适合你吗？

如果你符合以下情况，这套方案适合：

- 已安装阳台光伏（微型逆变器系统）
- 希望尽量用掉自己发的电，减少电量送回电网（如收益低或政策限制）

:::info 
- 如果需要扩展光伏，可以通过微储（SolidFlex 2000 / PowerFlex 2000）接入 
- 如果不需要扩展光伏，可选择 Eco 版本（SolidFlex 2000 Eco / PowerFlex 2000 Eco），降低成本  
:::

## 这套系统能帮你做什么？

简单来说，设备会自动帮你管理光伏电能：

👉 **家里优先用 → 多余存起来 → 减少送回电网**

你可以获得：

- ✅ 白天发的电，优先在家里使用  
- ✅ 用不完的电存进电池，晚上继续用  
- ✅ 减少低收益或受限的并网电量  

## 系统结构说明

系统包含以下几个部分：

- 阳台光伏组件（DC）
- 微型逆变器（DC → AC）
- Indevolt 微储
- 家庭负载（插座 / 空调 / 家电）
- 配电箱（电表 / 断路器）
- 电网

## 设备如何连接？


### 1️⃣ 阳台光伏接入

```
光伏板（DC）
   ↓
微型逆变器（DC → AC）
   ↓
接入 Indevolt 微储 Backup（旁路口）
```

> 💡说明：
> - 光伏须先经过微逆，再接入设备旁路口
> - 旁路口会自动识别为电源输入，将电能分配给负载或电池


### 2️⃣ 微储接入家庭电路

```
Indevolt 微储 GRID IN/OUT 口
   ↓
插入家中插座
```

> 💡说明：
> - 设备通过插座接入家庭电网（即并网）
> - 安装简单，无需复杂布线

## 系统是怎么工作的？

**🌞 白天（有光伏）**

1. 光伏发电 → 优先供家庭负载 + 电池充电
2. 剩余电量 → 才可能送入电网

**🌙 晚上（无光伏）**

1. 电池 → 供家庭负载
2. 电池不足 → 电网补充
