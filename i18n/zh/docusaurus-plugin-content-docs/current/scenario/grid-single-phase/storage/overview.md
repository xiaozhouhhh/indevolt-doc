# 纯储能系统（暂无光伏）

## 场景概述

该方案适用于**未配置光伏系统**的家庭或小型用电场景，主要依靠“电网 + 微储”实现能量管理与电费优化。

## 适用人群

如果您符合以下任一情况，该方案可能适合您：

- 未安装光伏系统
- 存在明显峰谷电价，希望降低用电成本
- 用电存在波动，希望提升用电稳定性
- 所在地区常有停电，需要基础备电能力

简单理解：使用电网充电，通过电池在不同时段释放电能，实现用电优化。

---

## 系统组成

**基础组成：**

- 微储
- 家庭电网

:::info 
- 如需扩展光伏，请选择标准版本（SolidFlex 2000 / PowerFlex 2000）
- 如无需扩展光伏，可选择 Eco 版本（SolidFlex 2000 Eco / PowerFlex 2000 Eco），Eco 版本不支持连接光伏，成本更低  
:::

**可选配置：**

在基础系统之上，可按需扩展以下能力：

- 是否可以监测用电（电表）
- 是否可以控制指定设备（插座）
- 是否可以断电供电（备电箱）
- 是否可以扩展容量（并机）

可以理解为：在基础储能系统上逐步叠加功能模块，实现能力升级。
