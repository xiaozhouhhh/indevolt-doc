# 微储单机

## 场景概述

这是最基础的并网单相储能接入方式，适用于当前未接入光伏的家庭。

- 微储通过家庭插座接入电网
- 与家庭配电系统并联运行
- 可为家庭负载供电，并与电网协同工作

简单理解：把电池接在插座上，参与家庭用电


## 系统配置

- 微储：单机运行
- 光伏：当前未接入（设备支持扩展，可按需选配光伏组件）

## 系统结构

微储通过插座接入家庭回路，与电网并联运行：

<img src={require("./img/scenario1.png").default} width="960"/>

- 微储 → 家庭插座 → 家庭回路 → 配电箱 → 电网
- 家庭负载共享微储与电网供电
- 关键负载可通过旁路口单独接入 

:::info
- 插座接入功率需符合当地法规限制（例如德国 800W 限制）
- 本场景基于输出功率**不超过 800W** 的典型使用情况
:::

## 系统能做什么？

在该配置下，系统主要用于降低电费与优化用电：

- 电池参与家庭供电，减少从电网取电
- 根据用能模式进行充放电调节
  - 自用优先：可自定义负载曲线，支持最多 48 个时间段的功率设置
  - 自定义充放计划：按设定时间自动充电或放电
- 支持后续扩展（电表 / 插座 / 备电等）

👉 详细用能策略说明请参考：[用能模式](../../../hardware/energy-mode/energy-mode.md)

## 场景特点

- 安装简单：无需改造配电系统
- 即插即用：适合已装修家庭
- 成本最低：基础入门方案
- 易扩展：可逐步升级功能
