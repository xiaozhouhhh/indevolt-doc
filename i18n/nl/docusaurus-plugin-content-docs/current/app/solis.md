# KeyID en KeySecret verkrijgen

Om SolisCloud met INDEVOLT te verbinden, moet u de KeyID en KeySecret van SolisCloud verkrijgen.

Raadpleeg de sectie **"2. API to Obtain Single Account Permission"** in de [officiële Solis-documentatie](https://doc.soliscloud.com/en/20.API%20documentation/01.SolisCloud%20Platform%20API%20Document.html) voor instructies.
