# App-Installation

:::info Voraussetzungen

Bevor Sie beginnen, stellen Sie bitte sicher, dass Ihr Gerät die folgenden Anforderungen erfüllt:

- Betriebssystem: iOS 13.0 oder höher; Android 5.0 oder höher.
- Verbindung mit einem funktionierenden WLAN oder mobilen Netzwerk.
- Ausreichend freier Speicherplatz.
- Ausreichender Akkustand oder schließen Sie Ihr Gerät an eine Stromquelle an.

:::
