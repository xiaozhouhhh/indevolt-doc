# Ein- und Ausschalten

Dieses Dokument beschreibt, wie Sie das Indevolt SolidFlex / PowerFlex 2000 Energiespeichersystem über die Gerätetaste ein- und ausschalten.

## Einschalten

In folgenden Situationen müssen Sie das Gerät möglicherweise einschalten:

- Nach der Erstinstallation oder einer Neuverkabelung
- Wenn das System manuell ausgeschaltet wurde und neu gestartet werden soll
- Nach einer längeren Stromunterbrechung

**Vorgehensweise:**

1. Stellen Sie sicher, dass das Gerät ausgeschaltet ist.
2. Halten Sie die Ein-/Aus-Taste etwa 2 Sekunden lang gedrückt.
3. Lassen Sie die Taste los. Das Gerät startet und die grüne Tastenbeleuchtung leuchtet auf.
4. Die blaue LED-Leiste zeigt eine Animation von der Mitte nach außen.
5. Nach dem Start zeigt die LED-Leiste die [entsprechende Statusanzeige](../overview/appearance-interface.md#led-anzeigen) des Systems an.

## Ausschalten

In folgenden Situationen wird empfohlen, das Gerät zunächst auszuschalten:

- Vor Wartungs-, Reinigungs- oder Verkabelungsarbeiten
- Wenn das System über einen längeren Zeitraum nicht verwendet wird
- Wenn die externe Stromversorgung sicher getrennt werden muss

**Vorgehensweise:**

1. Stellen Sie sicher, dass das Gerät eingeschaltet ist.
2. **Trennen Sie die PV-Eingänge und warten Sie mindestens 20 Sekunden**, damit keine Energieerzeugung mehr erfolgt.
3. Halten Sie die Ein-/Aus-Taste etwa 2 Sekunden lang gedrückt.
4. Die blaue LED-Leiste zeigt eine Animation von außen zur Mitte.
5. Lassen Sie die Taste los. Das System startet den Abschaltvorgang.
6. Sobald die Tastenbeleuchtung vollständig erloschen ist, ist das Gerät erfolgreich ausgeschaltet.

:::info

- Drücken Sie die Ein-/Aus-Taste während des Start- oder Abschaltvorgangs nicht wiederholt.
- Warten Sie nach dem Ausschalten, bis das Gerät vollständig heruntergefahren ist, bevor Sie Wartungs- oder Verkabelungsarbeiten durchführen.
- Wenn das Gerät längere Zeit nicht verwendet wird, empfiehlt es sich, es auszuschalten und von der externen Stromversorgung zu trennen.

:::
