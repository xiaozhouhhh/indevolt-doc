# So verbinden Sie sich mit GoodWe


## Schritt 1: SEMS-Konto erforderlich

Um auf die GoodWe SEMS API zuzugreifen, benötigen Sie ein SEMS-Konto (E-Mail-Adresse/Benutzername und Passwort).

Falls Sie noch kein SEMS-Konto besitzen, erstellen Sie zunächst eines.

---

## Schritt 2: Datenquelle konfigurieren

Klicken Sie auf „Add access“ und geben Sie die folgenden Informationen ein:

- Benutzername: E-Mail-Adresse des SEMS-Kontos
- Passwort (unverschlüsselt): Passwort des SEMS-Kontos

Nachdem Ihr Gerät online ist, kann es 5–10 Minuten dauern, bis die Seriennummer (SN) in der GoodWe-Geräteliste angezeigt wird.
