# Solax

Um Ihren Solax-Solarwechselrichter in die INDEVOLT App hinzuzufügen, benötigen Sie **tokenID**, **API Address** und **Registration Number** von SolaxCloud.

---

## So erhalten Sie tokenID und API Address

**Mit der Solax-App**:
1. Tippen Sie in der App unten auf dem Hauptbildschirm auf **Service** und wählen Sie auf dieser Seite **Third-party ecosystem**;
2. Wählen Sie **SolaxCloud API**;
3. Kopieren Sie die in der App angezeigte **token ID** und **API Address**.

   <img src={require("./img/third_party.png").default}  width="240"/>
   <img src={require("./img/solaxcloud_api.png").default}  width="240"/>
   <img src={require("./img/tokenid.png").default}  width="240"/>

**Mit Solax Web**:
1. Klicken Sie auf der Hauptseite in der linken Seitenleiste auf **Service**;
2. Klicken Sie auf **Third-party ecosystem**;
3. Hier können Sie Ihre **token ID** und **API Address** kopieren.

   <img src={require("./img/solax_web_third_party.png").default}/>
   <img src={require("./img/solax_web_tokenid.png").default}/>

:::note
Wenn keine Token-ID angezeigt wird, klicken Sie auf **Get**, um eine anzufordern, und kehren Sie dann zu Schritt 3 zurück, um sie zu kopieren.

   <img src={require("./img/get_tokenid.png").default} width="480"/>
:::

---

## So finden Sie die Registration Number

**Mit der Solax-App**:
1. Tippen Sie in der App unten auf dem Hauptbildschirm auf **Device**;
2. Ihre Wechselrichterliste zeigt die **Registration Number** an.

   <img src={require("./img/app_registration_number.png").default}  width="240"/>

**Mit Solax Web**:
1. Klicken Sie auf der Hauptseite in der linken Seitenleiste auf **Devices**;
2. In der **Devices List** finden Sie die **Registration Number**.

   <img src={require("./img/web_registration_number.png").default}/>
