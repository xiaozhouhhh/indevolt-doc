# So erhalten Sie KeyID und KeySecret

Um SolisCloud mit INDEVOLT zu verbinden, benötigen Sie die KeyID und das KeySecret von SolisCloud.

Bitte beachten Sie den Abschnitt **"2. API to Obtain Single Account Permission"** in der [offiziellen Solis-Dokumentation](https://doc.soliscloud.com/en/20.API%20documentation/01.SolisCloud%20Platform%20API%20Document.html), um die entsprechenden Anweisungen zu erhalten.
