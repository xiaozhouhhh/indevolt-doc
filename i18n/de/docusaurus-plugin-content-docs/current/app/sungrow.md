# So erhalten Sie den Appkey und den Secret key

Führen Sie die folgenden Schritte aus, um im Sungrow Developer Portal eine Anwendung zu erstellen und Ihren **Appkey** sowie **Secret key** zu erhalten.

1. Öffnen Sie https://web3.isolarcloud.eu/#/login und melden Sie sich mit Ihrem iSolarCloud-Konto an.

   <img src={require("./img/sungrow1.png").default} />

2. Klicken Sie oben rechts auf das <img src={require("./img/sungrow_icon.png").default} width="20" style={{verticalAlign: "middle"}}/> Symbol und wählen Sie anschließend **Developer Portal** aus.

   <img src={require("./img/sungrow2.png").default} />

3. Klicken Sie im [Developer Portal](https://developer-api.isolarcloud.com/#/home) oben rechts auf **Applications**, um die Seite **My applications** zu öffnen.

   <img src={require("./img/sungrow3.png").default} />

4. Klicken Sie auf **Create application**, um eine neue Anwendung zu erstellen.

   <img src={require("./img/sungrow4.png").default} />

5. Füllen Sie die Anwendungsinformationen aus:

   - **Application name**: Geben Sie den App-Namen (INDEVOLT) oder einen anderen leicht erkennbaren Namen ein, z. B. `My Home EMS`.
   - **Company name**: Geben Sie den Namen des in der App erstellten Zuhauses oder eine andere geeignete Bezeichnung ein, z. B. `My Home`.
   - **Application description**: Geben Sie eine kurze Beschreibung ein, z. B. `Data Integration`.
   - **Authorize with OAuth2.0**: Wählen Sie **No** aus.

   Klicken Sie anschließend auf **Create application**.

   <img src={require("./img/sungrow5.png").default} />

6. Kehren Sie zur Seite **My applications** zurück und klicken Sie bei der soeben erstellten Anwendung auf **View**.

   <img src={require("./img/sungrow6.png").default} />

7. Auf der Detailseite der Anwendung werden Ihr **Appkey** und **Secret key** angezeigt.

   <img src={require("./img/sungrow7.png").default} />
